# spa-lite

Lightweight and minimal dom helpers

## Features

- [x] apply data into dom based on dataset (`data-*`) attributes
  - [x] text
  - [x] disabled
  - [x] hidden
  - [x] value
  - [x] class
  - [x] href
  - [x] src
  - [x] alt
  - [x] title
  - [x] onclick
- [x] fetch and cache html template
